Orion 1.5.2

  • Install Orion
  • Make sure the jsp-examples work
  • Download lastest build by downloading following resource: http://www.mmbase.org:8070/testing/common/lastbuild/mmbase.war
Installing MMBase
  • Go to the directory %orion% and move the directory %orion%/default-web-app to %orion%/default-web-app.bak
  • Create the directory %orion%/default-web-app
  • Move into this %orion%/default-web-app directory
  • Type 'jar xvf %downloaded-mmbase-war% (there should now be a mmbase directory, a WEB-INF and a META-INF directory), assuming that jar is inside your path.
Updating Orion
  • Go to the directory %orion%
  • Remove the files xalan.jar and xerces.jar
  • Now copy (or create a symlink) the file's %orion%/default-web-app/mmbase/WEB-INF/lib/xalan.jar, %orion%/default-web-app/mmbase/WEB-INF/lib/xerces.jar, %orion%/default-web-app/mmbase/WEB-INF/lib/xml-apis.jar into the %orion% directory.
Running Orion
  • Orion can now be started the way it has always been started
  • Editors can now be found at the location: http://serverhost:serverpost/mmbase/edit/ (login: admin password: admin2k)
